# VinFix Web

Website first React app for VinFix.

## Run locally

```bash
cd web
npm install
cp .env.example .env
npm run dev
```

## Backend

The website expects the backend at:

```text
VITE_API_BASE_URL=https://autofixhelp-api.onrender.com
```

## Deploy

Deploy this `web` folder to Vercel, Netlify, or Render Static Site.
